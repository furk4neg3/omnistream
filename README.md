# OmniStream

OmniStream is a cloud-native real-time RAG platform that ingests streaming support ticket events, processes them, stores vector embeddings, and serves retrieval-augmented answers through an API.

## v1 Scope
- Simulated real-time support ticket ingestion
- Streaming pipeline
- Indexing into a vector store
- Query API with retrieval + RAG
- Terraform-managed infrastructure
- Dockerized services
- Kubernetes deployment target

## Services
- producer
- indexer
- query-api
- router-agent
- processing-agent
